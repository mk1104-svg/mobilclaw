# MobilClaw MVP Plan (v1.0)

## What is Included

Core Features:
- Launcher with AI suggestions
- Macro creation & execution
- Triggers: Time, location, notification, app open, screen state
- Actions: Send SMS, open app, toggle WiFi/Bluetooth, vibrate
- Constraints: Time range, location, app state, day of week
- Pattern Learning: Auto-detect daily routines
- Bill Detection: Identify bills in notifications

Testing:
- 22 unit + integration tests
- >80% code coverage
- Manual testing on 5+ devices

Documentation:
- User guide
- API docs
- Troubleshooting
- In-app help

## What is NOT Included

Smart home control (Phase 2)
Voice commands (Phase 2)
Wearable integration (Phase 4)
Cloud sync (Phase 4)
Enterprise features (Phase 5)

## Technical Requirements

Battery (idle): <2% drain

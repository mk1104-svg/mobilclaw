# MobilClaw

> One Device. Complete Control.

[![License: GPL-3.0](https://img.shields.io/badge/License-GPL%203.0-blue.svg)](LICENSE)
[![Status](https://img.shields.io/badge/Status-Alpha-yellow.svg)](https://github.com/mk1104-svg/mobilclaw)

MobilClaw transforms your Android phone into an intelligent command center. Control your device with powerful automations, learn your patterns, and never miss a bill again.

## 🎯 Features

- **🤖 AI-Powered Launcher** - Intelligent app suggestions based on your habits
- **🔧 Macro Automation** - Create powerful automations without coding
- **📊 Pattern Learning** - Discover your daily routines automatically
- **💰 Bill Detection** - Automatically identify bills in notifications
- **🎯 Location Awareness** - Triggers based on GPS and WiFi
- **🔐 Privacy-First** - All processing happens locally on your device

## 📲 Quick Start

### Requirements
- Android 12+ (API 31+)
- 100MB free storage

### Installation

**Option 1: Play Store**
Coming August 2026!

**Option 2: Build from Source**
```bash
git clone https://github.com/mk1104-svg/mobilclaw.git
cd mobilclaw
./gradlew build
```
